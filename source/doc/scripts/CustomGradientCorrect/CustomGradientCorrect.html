Multi Gradient Comparison

Scripts > EGo > Multi Gradient Comparison

Summary

Runs three different gradient-removal processes on independent clones of a single calibrated image and shows you the corrected output and the background model from each. Lets you pick the best one for the target without committing to a single tool.

What it does

  1. Clones the source so the original is never touched.
  2. Runs SpectrophotometricFluxCalibration (SPFC) on the clone — the prerequisite for MultiscaleGradientCorrection. SPFC uses the chosen telescope setup's sensor QE curve and filter parameters (broadband brand for LRGB/RGB; narrowband wavelength and bandwidth for SHO/HOO/…). Mono images run SPFC in single-channel mode driven by the FITS FILTER keyword.
  3. On three more clones of the SPFC-calibrated image, runs GradientCorrection (GC), MultiscaleGradientCorrection (MGC), and AutomaticBackgroundExtractor (ABE). Keeps both the corrected output and the gradient/background model from each.

Mono vs color

  • Mono: filter auto-detected from the FITS FILTER keyword.
  • Color: palette parsed from the image name (SHO, HOO, HSO, OHS, HOS, LRGB, RGB). If none found, pick one from the dropdown before running.
Customization. The telescope setup table, broadband filter brand curves, and narrowband bandwidth defaults live near the top of the script source. Edit them to match the curve names in your SPFC curve database.
