1 Description

[hide]

This script executes a chain of linear-stage processes on the active view, with two presets tuned for the targets the author shoots most: narrowband nebulae and broadband galaxies.

The chain consists of gradient correction, optional plate solve and SPCC for broadband, BlurXTerminator, NoiseXTerminator, and an optional star/starless split with StarXTerminator. Each step is reversible through PixInsight's Undo system and the chain halts on the first failure or cancellation.

Missing third-party processes (BXT, NXT, SXT, GradientCorrection) are detected at runtime and skipped with a console warning rather than aborting the chain.

2 Usage

[hide]

Open the script from Scripts > EGo > One-Click Linear Workflow. Pick the target type, toggle the steps you want, and click Run.

The active image window is the target. The image must be linear; running this on a stretched image will produce poor results.

Target type

Switches between the narrowband and broadband galaxy presets. Disables plate solve and SPCC checkboxes in narrowband mode.

Gradient correction

Runs GradientCorrection; falls back to AutomaticBackgroundExtractor if GC is not installed.

Plate solve

Launches the bundled ImageSolver script. Broadband only.

SPCC

Opens SpectrophotometricColorCalibration in interactive mode. Broadband only.

BlurXTerminator

Runs BXT with preset values for the chosen target type.

NoiseXTerminator

Runs NXT with detail 0.15 and denoise 0.85.

Star split

Runs StarXTerminator with stars=true and unscreen=true, producing both the stars-only and starless siblings of the input.

3 Workflow steps

[hide]

3.1 Narrowband preset

GradientCorrection -> BlurXTerminator (sharpen_nonstellar 0.85, sharpen_stars 0.20) -> NoiseXTerminator (denoise 0.85) -> optional StarXTerminator split.

3.2 Broadband galaxy preset

GradientCorrection -> ImageSolver dialog -> SPCC dialog -> BlurXTerminator (sharpen_nonstellar 0.75, sharpen_stars 0.25) -> NoiseXTerminator (denoise 0.85) -> optional StarXTerminator split.
