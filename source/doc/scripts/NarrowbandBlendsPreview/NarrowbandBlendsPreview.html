NarrowbandBlendsPreview

By Evan

Live grid preview of every narrowband blend recipe in the EGo catalog: static palettes (Classic Hubble, HOO, Foraxx, ...), dynamic PIP-gated blends, and spatial split blends. Pick a tile, then open it in PixelMath or apply it to the active view. [more]

1 Description

[hide]

Each blend is rendered on auto-STF'd downsampled copies of the selected channel views, so you can compare every palette side-by-side at a glance. Click a tile to select; the actions below operate on the selected recipe at full resolution.

Recipes use single-letter symbols (H, O, S, plus L, R, G, B for LRGB-flavored ones) bound to actual view IDs via PixelMath's Symbols field. Re-targeting a saved icon to a different image set is a one-field edit later.

Dynamic blends use per-pixel gates derived from the channels themselves (e.g. gO = O^~O) for smooth maskless transitions between palettes. Spatial blends use X() / Y() to weight one palette against another across the frame.

2 Usage

[hide]

Open from Scripts > EGo > NB Blends Live Preview. Pick at least Ha and OIII views (and SII / L / R / G / B as needed by individual recipes), click Generate / Refresh Previews, then click a tile and either Open in PixelMath or Apply to Active.

Channel image selectors

ViewLists for H (Ha), O (OIII), S (SII), L, R, G, B. Ha and OIII are required; the others are used only by recipes that reference them.

Generate / Refresh Previews

Downsamples the selected channels (longest side capped at 220 px), runs every blend against the downsampled clones, applies a per-channel Auto-STF, and renders each result into its tile. Tiles that need channels you haven't selected show Needs: X,Y.

Open in PixelMath

Launches a configured PixelMath dialog at full resolution with the recipe's expressions and symbol bindings already filled in. Drag the PixelMath title bar to the workspace to save the recipe as a process icon.

Apply to Active

Runs the selected blend in place on the currently active view. Destructive; use Edit > Undo to revert.
