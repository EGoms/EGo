StretchComparison

By Evan

Clones the target image once per enabled stretch method and applies each to a separate clone for side-by-side comparison. [more]

1 Description

[hide]

Methods covered: Auto-STF baked into HistogramTransformation, MaskedStretch, ArcsinhStretch, GeneralizedHyperbolicStretch, and AdaptiveStretch. The original is never modified; output windows are suffixed with the method tag (_STF, _MS, _AS, _GHS, _AdS) and remain open in PixInsight.

Per-method tuning constants live at the top of the script source - edit them to match your preferred defaults.

2 Usage

[hide]

Open from Scripts > EGo > Stretch Comparison. Choose the source view, enable the methods you want to compare, and click Run.

Source view

Any mono or color view. Must be linear.

Method toggles

Independent on/off for STF, MaskedStretch, ArcsinhStretch, GHS, and AdaptiveStretch.
