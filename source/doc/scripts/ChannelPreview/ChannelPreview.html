Channel Preview

Scripts > EGo > Channel Preview (all spaces)

Summary

One-shot generator of every common channel decomposition of a color image, plus an optional tiled contact sheet for at-a-glance comparison. Designed for narrowband palette experimentation (which channel carries the cleanest Ha vs OIII signal? which RGB combine gives the best a* / b*?) and for diagnosing broadband color casts.

Color spaces extracted

SpaceChannelsNotes
RGBR, G, BIdentity for color images.
CIE L*a*b*L, a, b Perceptual lightness + opponent color axes; a* (green-red) and b* (blue-yellow) are useful for narrowband palette swaps.
CIE L*c*h*L, c, h Lab in cylindrical form; c is chroma, h is hue angle.
HSVH, S, VHue + saturation + value.
HSIH, S, IIntensity variant of HSV; the intensity channel is a common luminance proxy.
CIE XYZX, Y, Z Linear tristimulus components.

Inputs

  • Source view must be a color image. The dialog enforces this.
  • Pick which spaces to extract. Sensible defaults: RGB, Lab, HSI.
  • Optionally also create inverted channels (handy when the interesting signal is in a low-value channel).
  • Optionally generate a tiled contact sheet. Defaults: 384 px tile size, 3 columns. The sheet is an 8-bit RGB image you can save as PNG.

Output naming

Each extracted channel becomes an image window named <sourceId>_<space>_<channel>. The contact sheet, if requested, is <sourceId>_contactSheet. Duplicates are auto-suffixed with _1, _2, … so re-running on the same source doesn't clobber previous outputs.
